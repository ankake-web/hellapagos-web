# 背景・タイトル挿絵の実画像（任意・差し替え用）

ここに画像を置くと、`manifest.ts` が自動検出してアプリへ反映します（無ければCSS/SVGのまま）。

| ファイル名 | 反映先 | 推奨仕様 |
|---|---|---|
| `scene.webp`（または .png/.jpg） | アプリ全体の背景（`Backdrop`） | 2560×1440・横長。中央上部は控えめ・やや暗め（UIが乗る） |
| `hero.webp`（または .png/.jpg） | ホーム画面のヒーロー挿絵（`HeroArt`） | 約 1200×675（16:9）・透過不要 |

プロンプトは `docs/asset-prompts.md` の「背景・タイトル」を参照。複数形式を置いた場合はどれか1つが使われます（1種類だけ置くこと）。
